# 高德地图API+VUE简单实现

## 获取高德地图API key 
> 高德开放平台：https://lbs.amap.com/

## Sept1：安装依赖
```
yarn install
```
## Sept2: 修改key
> 项目src --- config --- config.js
## Sept3: 启动
```
yarn serve
```
